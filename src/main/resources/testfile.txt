	В текущее время существует два различных подхода к описанию диффузии. Первый подход заключается в построении стохастического дифференциального уравнения, которое описывает движение системы.
	Второй подход — составление уравнения Фоккера-Планка и его решение.
	При этом существуют две различные формы уравнения Фоккера-Планка. Представляет интерес связь между стохастическим дифференциальным уравнением и уравнением Фоккера-Планка, составленными для одной системы, и связь между различными формами уравнения Фоккера-Планка, поскольку это позволит гибче выбирать метод для решения конкретной задачи.
	Численное интегрирование стохастических дифференциальных уравнений имеет особенности, которые необходимо выделить для решения прикладных задач.
